# custom-vue-full

> Vue.js 프로젝트

## Build Setup

``` bash
# 개발 의존 모듈 설치
npm install

# 웹 서버(localhost:8080)로 구동
npm run dev

# 배포를 위해 압축 후, 빌드
npm run build

# 배포를 위해 압축 후, 빌드 (번들링 리포트 포함)
npm run build --report
```
